## 🚀 Getting Started
1. **Clone the repository**  
   ```bash
   git clone https://github.com/Darkfalcon2804/Cplusplus.git
   ```
2. **Navigate to the project folder**  
   ```bash
   cd Cplusplus
   ```
3. **Compile and run a C++ program**  
   ```bash
   g++ filename.cpp -o output
   ./output
   ```
